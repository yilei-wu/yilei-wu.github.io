# 我的第一篇博客

欢迎来到我的博客！这是我的第一篇文章。

## 为什么要写博客？

我很喜欢看别人的分享，无论是来源于生活圈内的朋友，还是学术圈的前辈。这些分享总是给了我很多的启发。这里我实在想要分享两个我觉得对我帮助很大的博客:
- [Chiyuan Zhang](https://freemind.pluskid.org/)
- [Lei Mao](https://leimao.github.io/blog/)
我印象里我不止一次的向我的朋友分享过并兴奋的复述了其中的一些内容。我不知道被我安利过的朋友是否像我一样欣赏他们，但时至今日，我还是会时不时的check out这些博客看看有没有新的内容。
